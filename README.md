# my_learning_record
このリポジトリは、Udemy講座「【世界で91万人が受講】基礎から理解し、Pythonで実装！機械学習26のアルゴリズムを理論と実践を通じてマスターしよう」（作成者：株式会社CODOR（大橋亮太）・SuperDataScience Team）で学習した内容をもとに、各章で実装したコードやアウトプットをまとめたものです。

機械学習の基本概念（前処理・単回帰分析・重回帰分析・分類・クラスタリングなど）をPython（pandas, scikit-learn, matplotlibなど）を用いて実装・可視化し、理論と実装の両面から理解を深めることを目的としています。


⇩ファイルの説明

● Simple_regression_analysis.ipynb

単回帰分析を学ぶために作成したコード。

●multiple_regression_analysis.ipynb

重回帰分析を学ぶために作成したコード。

●polynomial_regression_analysis.ipynb

多項式回帰を学ぶために作成したコード。

● support_vector_regression.ipynb

サポートベクトル回帰（SVR）を学ぶためのコード。

● decision_tree_regression.ipynb

回帰木を学ぶためのコード。

●random_forest.ipynb

ランダムフォレストを学ぶためのコード。

● LogisticRegression1.ipynb

ロジスティック回帰を用いて、複数の指標からアーティストが紅白歌合戦に選出されるかどうかを予測する分類モデルを作成したコード。

●LogisticRegression2.ipynb

ロジスティック回帰を用いて、今年最もいいね数が多かったYouTubeのMVと、今年最も再生数が多かったSpotifyの楽曲という2つの指標から紅白選出を分類し、決定境界を可視化したコード。

●KNN.ipynb

K近傍法（KNN）を用いて、今年最もいいね数が多かったYouTubeのMVと、今年最も再生数が多かったSpotifyの楽曲という2つの指標から紅白選出を分類し、決定領域を可視化したコード。

●Support_vector_machine.ipynb

サポートベクトルマシン（SVM）を用いて、インターネット利用率と第三次産業割合の2つの指標から国の経済発展度（1人当たりGDPが30000以上か否か）を分類し、決定境界を可視化したコード。

●Kernel_svm.ipynb

カーネルサポートベクトルマシン（Kernel SVM）を用いて、年齢と推定年収の2つの指標から商品の購買有無を分類し、非線形な決定境界を可視化したコード。

●Naive_bayes.ipynb

ナイーブベイズ（GaussianNB）を用いて、インターネット利用率と第三次産業割合の2つの指標から国の経済発展度（1人当たりGDPが30000以上か否か）を分類し、決定領域を可視化したコード。

●Desicion_Tree.ipynb

分類木を用いて、インターネット利用率と第三次産業割合から国の経済発展度を分類し、他のモデルよりも細かい決定境界がどのように形成されるかを確認するコード
