●Simple regression analysis.ipynb


🎯 学習目的

単回帰分析（Simple Linear Regression）の理解と実装を目的として作成しました。

統計的手法を用いて、1つの説明変数（人口）から1つの目的変数（GDP）を予測するモデルを構築し、口とGDPの間にどのような関係があるのかを可視化しました。


💡 分析内容

データ：47都道府県の人口とGDPデータ（pref_population_gdp.csv）

モデル：線形回帰（Linear Regression）


手順：

欠損値を中央値で補完

学習データとテストデータに分割

線形回帰モデルを学習

回帰直線と実測値（散布図）をプロット


📊 結果と考察

全体として「人口が多いほどGDPが高い」という正の相関が見られました。

しかし、東京都など人口に対してGDPが極端に高い地域があり、単回帰モデルでは十分に説明できないことも確認できました。

●Multiple Regression Analysis.ipynb


🎯 学習目的

本ノートブックは、重回帰分析（Multiple Linear Regression）を理解し、実データを使って実装できるようになることを目的として作成しました。

複数の説明変数を同時に扱うことで、「GDPに影響を与える要因を統計的に特定する」ことを狙いとしています。


💡 分析内容

🔹 使用データ

47都道府県に関する以下の 8 つの特徴量（説明変数）を用いました：

Population（人口）

Number of establishments（事業所数）

Proportion of people working in primary industries（第一次産業従事者割合）

Proportion of people working in secondary industries（第二次産業従事者割合）

University enrollment rate（大学進学率）

population density（人口密度）

Average annual income（平均年収）

GDP（目的変数、または別変数の場合は適宜調整）

🔹 モデル

statsmodels OLS（最小二乗法）

add_constant() により切片を追加した一般的な重回帰モデルを構築


📝 手順

データの読み込み・基本統計量の確認

目的変数 y と 説明変数 X の分離

切片追加（add_constant）

statsmodels による重回帰分析の実行

p値・係数・R²の確認

p値にもとづく変数選択（フィーチャーセレクション）

不要な変数を除去し、人口・事業所数・平均年収 のみで再学習

予測値と実測値の比較


📊 結果と考察

🔹 p値による変数選択の結果→ 最終的に x1（人口）・x2（事業所数）・x7（平均年収） の3変数だけが統計的に有意（5%未満）で残った。

🔹 モデル全体の性能

R² = 0.974
→ GDPの変動の約 97.4% を説明できる非常に高い精度。

Adjusted R² = 0.969
→ 説明変数が多い場合でも過学習は起きておらず、モデルは安定。


🔹 最終モデル（x1・x2・x7 のみ）による考察

人口が多い地域ほど GDP が高い
→ 予想通り強い正の関係。

事業所数も GDP と強い正の相関
→ 経済活動の量を反映しているため妥当。

平均年収（または x7）が有意
→ 労働生産性や産業構造の違いが GDP に影響していると解釈できる。

一方で、第一次産業、第二次産業、大学進学率、人口密度などは今回のモデルでは「単独では有意ではなかった」ため除外。
