🎯 学習目的

単回帰分析（Simple Linear Regression）の理解と実装を目的として作成しました。
統計的手法を用いて、1つの説明変数（人口）から1つの目的変数（GDP）を予測するモデルを構築し、口とGDPの間にどのような関係があるのかを可視化しました。


💡 分析内容

データ：47都道府県の人口とGDPデータ（pref_population_gdp.csv）
モデル：線形回帰（Linear Regression）


手順：

　欠損値を中央値で補完

学習データとテストデータに分割
線形回帰モデルを学習
回帰直線と実測値（散布図）をプロット


📊 結果と考察

全体として「人口が多いほどGDPが高い」という正の相関が見られました。
しかし、東京都など人口に対してGDPが極端に高い地域があり、単回帰モデルでは十分に説明できないことも確認できました。
この結果から、今後は「産業構造」「企業数」「一人当たり生産性」など、複数の説明変数を用いた重回帰分析に発展させる余地があると分かりました。
